## About Fb-clone

This project is kind of replica of facebook build using laravel 8, vue js and tailwind css

## Installation Commands
larvel new projectname ---- to create laravel project 
php artisan migrate ---- to run the migration tables
composer require laravel/passport ---- Passport Package added to the composer json dependency package manager
composer require laravel/ui ---- laravel ui is setup
php artisan ui vue ---- to generate sample component in the app.js
php artisan ui:auth ---- Authentication Scaffolding
npm install ---- npm package manager and packages for vue
npm run watch ---- dev watcher for ui code changes
npm install vue ---- vue front end 
npm install vue-router --save-dev ---- vue router for router
npm install tailwindcss --save-dev ---- tailwindcss part of the frontend
npx tailwind init ---- creates tailwind.config.js file
composer require laravel/legacy-factories ---- use factory()
